About

What is this page for?

It provides a way to download your EVE Online character's market transaction in CSV format, both in the current and the "legacy" gestalt (as in the days of the now deprecated XML API). It also offers a very specialised way of retrieving contract data (single item sells from public contracts - the kind you might use to sell researched blueprints). While you're at it, you can look at some pretty graphs.

But why?

Back in the days of the XML API I was using its endpoints very extensively to monitor my market transactions and was sitting on a huge stash of market data reaching back years. When the switch from the XML API to ESI Swagger API happened I had the options to either discontinue my past diligence or find a way to deal with the new challenge, figure out how to make the (more complex) ESI API work and how to transform its new outputs into a form that I could easily pipe into my existing data.

Why R?

R was known to me as being good for data acquisition and transformation, and since I already had some practice in it, it seemed the right choice. I also had been wanting to try out the Shiny package for quite some time, so basically it is a "two birds one stone" story.

What is EVE?

EVE is a MMO (massive multiplayer online) game from CCP Games. It is set in humanity's distant future among the stars and is one of the few true and oldest sandbox games still online (release 2003).

It has a reputation of being continually taken very seriously by its players, featuring ruthless leaders of alliances of tens of thousands of players, cut-throat diplomacy, back room deals, intricate espionage, gigantic space battles that puzzle the news for their destruction of wealth and a living virtual economy that economists keep writing papers about (this is how I learned about it, too). A historian wrote a book about its empires of the first decade. Plus the regular confusion if this is actually still a game, a lifestyle, "real" or a cult depends on the person you ask.
Oh, and did I mention its third party tool development community because CCP is so awesome to make basically every ingame data accesible via an API?

Let's get to the real question: Is this a honeypot?

No. You can look up the code of the app on my github repo. But it could be one, and being paranoid about this shows you have already understood EVE's nature.

How can you stop it from being able to pull data? See all third party applications that you gave permissions to on this page. Look for "RShiny Wallet_cloud". Then revoke permissions.

Are you too paranoid to use it? You can run a local version of the app! Just check it out from my github repo mentioned above.

What is Shiny?

It's a package for R and a framework for building interactive web apps. It makes pretty, open source dashboard applications.
